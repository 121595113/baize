# element-ui管理后台脚手架

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for Publish
>development : `yarn buildDev`

>uat : `yarn buildUat`

>production : `yarn build`

### Lints and fixes files
```
yarn lint
```
